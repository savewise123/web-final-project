주요 기능

1. 메인 페이지 (`/`)
    - 글 목록 조회
2. 상세 페이지 (``)
    - 글 상세 조회
    - 추천 수 조회
    - 추천 기능 (로그인 후 가능)
    - 댓글 수 조회
    - 댓글 목록 조회
    - 댓글 작성  (로그인 후 가능)
    - 댓글 수정 (로그인 후 가능)
    - 댓글 삭제 (로그인 후 가능)
3. 글 작성 페이지 (``)
4. 글 수정 페이지 (``)
5. 회원가입 페이지
    - 이메일
    - 닉네임
    - 비밀번호
    - 비밀번호 확인
    - 유효성 검사
6. 로그인 페이지 
    - 이메일
    - 비밀번호
    - 유효성 검사
7. 마이 페이지
    - 프로필 이미지 업로드
    - 닉네임 변경
